# 모두의 근육

Next.js(App Router) 기반 PWA 웹앱 MVP. 시각적 피드백 기반 체형·운동 포인트 안내(의학적·전문가 판정 아님).

## 기술 스택

- Next.js 14+ App Router
- TypeScript
- TailwindCSS
- 카드 기반 미니멀 UI, 한국어

## 폴더 구조

```
momuscle/
├── app/
│   ├── layout.tsx       # 루트 레이아웃, PWA 메타/manifest 링크
│   ├── page.tsx         # "/" 온보딩 (성별 + 사진 3종 업로드)
│   ├── globals.css
│   ├── survey/
│   │   └── page.tsx     # "/survey" 설문 6문항
│   └── result/
│       └── page.tsx     # "/result" 결과 4카드 + 다시 분석하기
├── lib/
│   ├── storage.ts       # localStorage 키/타입/헬퍼 (momuscle_*)
│   └── ruleEngine.ts    # 성별+설문 기반 결과 연산 (남/여 분기)
├── public/
│   ├── manifest.json    # PWA manifest
│   └── icons/
│       ├── icon-192.png  # placeholder
│       └── icon-512.png  # placeholder
├── package.json
├── tsconfig.json
├── tailwind.config.ts
├── next.config.js
└── postcss.config.js
```

## 실행 방법

```bash
cd momuscle
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속.

## 사용자 단계 가이드 (어떤 버튼을 누르면 되는지)

### 1. 홈("/") — 업로드/온보딩

1. **성별 선택**: **남자** 또는 **여자** 버튼 중 하나를 누릅니다.
2. **상체 정면**: "사진 선택" 버튼을 눌러 상체 정면 사진 1장을 고릅니다. (촬영 가이드: 얼굴 제외, 밝은 조명, 같은 거리·각도, 타이트한 옷, 단순한 배경, 팔 자연스럽게.)
3. **상체 등(후면)**: "사진 선택" 버튼을 눌러 상체 등 사진 1장을 고릅니다. (견갑·광배 라인이 보이도록.)
4. **하체**: "사진 선택" 버튼을 눌러 하체 사진 1장을 고릅니다.
5. 성별 + 3장 모두 입력되면 **"다음"** 버튼이 활성화됩니다. **"다음"**을 누르면 설문 페이지("/survey")로 이동합니다.

### 2. 설문("/survey") — 6문항

1. **Q1 목표**: 벌크업 / 다이어트 / 균형 중 하나를 누릅니다.
2. **Q2 운동 경력**: 0~6개월 / 6~24개월 / 2년+ 중 하나를 누릅니다.
3. **Q3 주 운동 횟수**: 1~2 / 3~4 / 5+ 중 하나를 누릅니다.
4. **Q4 약점 체감 부위**: 어깨·등·가슴·팔·하체·코어 중 해당하는 것을 **복수 선택**합니다(여러 개 눌러도 됨).
5. **Q5 선호 운동 스타일**: 머신 위주 / 프리웨이트 위주 / 섞어서 중 하나를 누릅니다.
6. **Q6 선호 결과**: "커 보이기(볼륨)" / "선명해 보이기(분리도)" / "비율 좋아 보이기(실루엣)" 중 하나를 누릅니다.
7. **"결과 보기"** 버튼을 누르면 결과 페이지("/result")로 이동합니다.

### 3. 결과("/result")

- 상단에 성별·목표·운동 횟수 요약이 보입니다.
- **결과 카드 1**: 체형 타입 (프레임형/볼륨형/라인형/하체강점형/밸런스형 중 하나).
- **결과 카드 2**: 지금 키우면 폼이 제일 빨리 달라지는 포인트 TOP2.
- **결과 카드 3**: 부위별 추천 운동(각 3개) + 1줄 이유.
- **결과 카드 4**: 8주 루트 요약(주 3~4회 기준).
- **"다시 분석하기"** 버튼을 누르면 로컬 데이터가 초기화되고 홈("/")으로 돌아갑니다.

### PWA — 홈 화면에 추가 (모바일)

- Android: Chrome에서 사이트 열기 → 메뉴(⋮) → "홈 화면에 추가".
- iOS: Safari에서 사이트 열기 → 공유 → "홈 화면에 추가".

아이콘은 `public/icons/`의 placeholder를 실제 앱 아이콘으로 교체하면 됩니다.
